# üìù About the Project

### General Description
Ecommerce API created as part of CoderHouse backend course project.

<hr> 

### üìù User Stories

1) A RESTful API will be implemented with the verbs GET, POST, PUT and DELETE to fulfill all the necessary actions.
2) It will contain the necessary routes that allow listing existing products, entering new products, deleting and modifying their details, as well as interacting with the shopping cart.

3) <ins> CRUDs: </ins> 
    - Products
   - Users.

4) <ins> MongoDB: </ins> 
   - Products: The entered products will be stored in a MongoDB database.
   - Users: The user will be able to register their access credentials (email and password) to later be able to access their account. These credentials will be saved on the database.

5) <ins> JWT: </ins> 
   - Authorized entry mechanism to the system based on JWT (Json Web Token).
   - The client will have an active user session with a configurable expiration time.
    - JWT token lifetime can be configured

6) <ins> Architecture: </ins> 
    - The server architecture will be based on layers (MVC)

7) An email will be sent to a configurable box, for each new user registration and with each purchase order generated.

# üîß How to run this Project?

### *Option 1:*
1. Get the URL from the green button on the top right corner of the `main` branch from this repository ![CodeGreenButton](https://user-images.githubusercontent.com/51333463/155273747-ddb43a67-477e-4e8f-813a-16f8c93eac52.png)
2. Choose a path on your computer
3. Execute the following command on your Terminal: `git clone https://github.com/arahualte/BackendAPI_Ecommerce.git`
4. On the project folder, execute `./gradlew bootRun` on your terminal
